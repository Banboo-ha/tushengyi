# UI_v1 切图清单与命名规范

前端图片统一放在：

`app/static/h5/assets/ui_v1/`

浏览器访问路径对应为：

`/h5-static/assets/ui_v1/`

## 命名规范

- 文件名统一使用小写英文、数字和中横线，例如 `home-hero-product.png`。
- 不使用中文、空格、括号、特殊符号，避免服务器部署后路径兼容问题。
- 图标类优先 `svg`，如果设计工具只方便导出位图，可以用透明背景 `png`。
- 插画、风格缩略图、横幅类优先 `webp`，也可以用 `png`。
- 同一个资源只保留一个主文件名，不额外加 `@2x`；如果导出高清图，直接覆盖同名文件。
- 用户上传图、模型生成图、作品封面不属于固定切图，不放在这个目录。

## 不需要固定切图

- 用户上传的产品图、参考图：保存到业务上传目录，不放在 UI_v1 切图目录。
- 模型生成的海报结果图：由后端生成并保存，不放在 UI_v1 切图目录。
- 作品库中的作品封面：使用生成图或版本图，不需要单独切。
